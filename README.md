## Опис роботи

У ході виконання лабораторної роботи було створено веб-сторінку мовою HTML5.
Сторінка містить основні елементи HTML-розмітки та відповідає вимогам стандарту HTML5.

### Використані елементи:

- Теги `<html>`, `<head>`, `<body>`
- Мета-теги `<meta>` (charset, keywords, description)
- Заголовки `<h2>`, `<h3>`
- Абзаци `<p>`
- Списки `<ul>`, `<ol>`
- Зображення `<img>`
- Гіперпосилання `<a>`
